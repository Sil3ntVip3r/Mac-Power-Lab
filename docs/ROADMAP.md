# MacPowerLab Roadmap

## Milestone 1 — Runtime profiles and independent logging

Versioned runtime settings, strict persistence, live/default/balanced/low/custom
profiles, independent UI/collector/log cadences, Live-only mode, conservative
ordinary niceness, safe session restarts, CLI/API/SwiftUI controls, and complete
session provenance.

## Milestone 2 — Help and sensor trust

Getting Started, FAQ, contextual metric information, provenance, accuracy,
normal guidance, warning help links, and an advanced terminology glossary.

## Milestone 3 — Visual history

Bounded live ring buffers, downsampled history, time-series graphs, meaningful
contract-relative gauges, and component/application share charts.

## Milestone 4 — Long-term local history

Session index, daily/monthly aggregates, battery-health trends, charger/cable
labels, application energy history, retention controls, and comparison views.

## Milestone 5 — Privacy foundation for optional community data

A versioned consent and export-preview system, on-device anonymization, a threat
model, and deletion/revocation controls. Upload remains absent until this design
is explicitly approved.
